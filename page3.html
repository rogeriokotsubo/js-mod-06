Exercício 3


Utilizando a mesma estrutura de 3 arquivos (HTML+CSS+JS):

Crie uma página web que contenha um alarme no qual o usuário define os minutos (máximo 59 minutos) e segundos e clica em um botão que inicia a contagem regressiva, considerando:

  1. Utilizar o HTML select para seleção dos minutos e segundos;
  2. Criar um botão que inicia a contagem regressiva que ao ser clicado este se torna um botão de desarmar alarme;
  3. O contador regressivo mostra o tempo a cada segundo;
  4. Quando faltar menos de 5% do tempo inicial mudar a interface de contagem para informar que o tempo está acabando;
  5. Ao término da contagem, o tempo deve parar de ser decrementado e deve-se disparar um alarme (som de alarme) informando que o tempo acabou;
  6. Se o usuário clicar no botão de desarmar alarme, a contagem deve ser parada e, caso o alarme esteja tocando, o som deve parar, transformando o botão para que possa armar o alarme novamente.
