Exercício 1


Utilizando a mesma estrutura de 3 arquivos (HTML+CSS+JS):

Crie uma página web que 'arme uma bomba' (imagem de uma bomba) que 'exploda' em 10 segundos, considerando:

  1. Utilize o setTimeout para 'armar a bomba' (bomba com pavil aceso);
  2. Caso o usuário clique na bomba, utilize o clearTimeout para desarmar a bomba (bomba com pavil apagado);
  3. Caso a bomba não seja desarmada, ao terminar o tempo, a bomba deve explodir e um som de explosão deve ser tocado;
